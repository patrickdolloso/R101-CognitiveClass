## Video 2.1: Arrays and Matrices
### Arrays
* vector of elements of the same datatype
* can be multi dimensional as well 
```R
# create a vector
greet_vector <- c("Hi","Hello","Sup","Howdy","Greetings","What's Up","Salut")
# convert to vector
greet_array <- array(greet_vector, dim=c(4,3))

```
